## 3. バイアス要因の仮説（断定しない表現）

### 仮説1: 代理変数による緩和

**観察:**
- 上位特徴に雇用期間、貯蓄額などが含まれる
- これらは年齢・性別と相関する可能性がある

**考えられる説明:**
- 雇用期間が長い → 信用度高い（年齢と相関）
- 貯蓄額が多い → 信用度高い（年齢・性別と相関）
- モデルが年齢・性別より「実質的な信用指標」を重視
- 結果として、直接的なバイアスが緩和される

**注意:** 代理変数が必ずしも因果関係を示すとは限らない

### 仮説2: データの質

**観察:**
- German Credit Dataは1990年代のドイツのデータ
- 元データでのグループ間差が小さい

**考えられる説明:**
- 当時のドイツの与信審査が比較的公平だった可能性
- データ収集時に既に一定の公平性配慮があった可能性
- サンプル選択バイアスの可能性（公平なケースのみ収録）

**注意:** 歴史的背景の検証が必要

### 仮説3: モデルの複雑度

**観察:**
- XGBoostの設定: max_depth=6, n_estimators=100
- 1000サンプルに対して適切な複雑度

**考えられる説明:**
- 過学習していないため、偏った相互作用を学習しない
- 正則化により、保護属性への過度な依存が抑制
- 結果として、バイアスが小さくなる

**注意:** より大きなデータセットでは異なる結果の可能性

## 5. 結論（暫定）

本分析では、以下の可能性が示唆された：

1. ✅ 代理変数（雇用期間、貯蓄額など）が保護属性の情報を
   適切に代替し、直接的なバイアスを緩和している **可能性**

2. ✅ データの質が高く、元々バイアスが小さい **可能性**

3. ✅ モデルの適切な複雑度が過学習を防ぎ、バイアスを
   抑制している **可能性**

**重要:** これらは仮説であり、更なる検証が必要である。
断定的な因果関係を主張するには、追加の実験と分析が求められる。
